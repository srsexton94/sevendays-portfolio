Project Timeline

Here are each of the projects completed for this challnege - click to see them live! My code is publically available on my GitHub.

Day 1: Holiday Lights

For the first day of this project I decided to code a fun, little, front-end JS project where the user can display a variety of holiday twinkle lights.

Day 2: Budget Tracker

For the second day, I tried my hand at my very first chrome extension - a budget tracker where users can be alerted when they reach their set spending goal.

Day 3: "To-Done!" Task Manager

The third day brings us a task manager application with a fun, home-desk UI: where the user can input their tasks via sticknote & track them on their notepad.

Day 4: Моя Русская Клавиатура

I next coded a typing practice application, with an apple-inspired interface, where users are able to practice their Russian typing skills. Удачи!

Day 5: Pomodoro Study Buddy

For the fifth day I decided to engage my time-managment obsession and make my very own Pomodoro clock application with an audio indicator at session end.

Day 6: "Right as Rain" Weather App

Next up I decided to make a Weather Forecast app! Utilizing a 3rd party API (OpenWeatherMap) and mobile-first design, the app reports min/max/current temps and a descriptive forecast.

Day 7: #7Days7Websites Portfolio!

Lastly, I concluded this project by crafting this portfolio you're viewing right now! Built from scratch with bootstrap snippets to collect and display each of these projects!
